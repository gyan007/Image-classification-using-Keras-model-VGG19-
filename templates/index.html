Image Classifier

The Image Classifier project is a web-based application designed to predict the category of a given image using deep learning models. Built using Flask for the backend and VGG19 (a pre-trained convolutional neural network) for image classification, this project leverages TensorFlow for model loading and prediction. The frontend is styled with modern CSS for an intuitive and interactive user experience.
